# WebSocket Multichat Project

**Учебный проект** для знакомства с протоколом WebSocket, распределенной обработкой сообщений и интеграцией мессенджеров.

Демонстрирует, как объединить чаты из разных платформ в единое пространство с помощью WebSocket-сервера и клиентов.

![](/docs/img/communication.png)

## Основной функционал
- Обмен текстовыми сообщениями между несколькими WebSocket-клиентами в реальном времени;
- Центральный WebSocket-сервер, который получает, фильтрует и маршрутизирует (если прошла фильтрация) сообщения между всеми подключенными клиентами;
- Возможность легкого добавления новых клиентов для других платформ.
